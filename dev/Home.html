01

Scanner Starten

Navigieren Sie zum Scanner und halten Sie Ihren Barcode bereit. Das System erkennt automatisch die Artikelnummer und initiiert einen synchronisierten Datenbankabgleich.

02

Hybrid Search Logic

Unsere v1.3 Engine nutzt eine Kombination aus OpenFoodFacts Live-Daten und lokaler SQL-Integrität. Wenn ein Artikel nicht direkt gefunden wird, übernimmt der fuzzy-gewichtetete Levenshtein-Algorithmus (78% Wort / 22% Edit-Distanz).

03

Terminal Interface

Erleben Sie Echtzeit-Feedback in unserem Cyber-Terminal. Von der Matrix-Datenstrom-Visualisierung bis hin zur automatischen Mengen-Diskrepanz-Prüfung – alles für maximale Transparenz.

04

Statistik & Analyse

Überprüfen Sie Ihren Lagerbestand in der Statistik-Ansicht. Alle Daten werden über einen automatisierten JTL-Wawi Export synchronisiert und clientseitig verarbeitet.

05

Live Cloud Sync

Verbindung konfigurieren

Geben Sie Ihre Cloudflare Tunnel URL ein, um eine Live-Verbindung zu Ihrem JTL-Wawi herzustellen.
